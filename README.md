## This is for Practice

###### Git Basic Commands


- git add - code will go from working dir to Staging area
- git commit -m , code will go from Staging area to Local repo
- git push - code will go from Local repo to remote repo
- git status

###### Advanced Commands

|Command|Description|
|:---:|:---|
|git add|code will go from working dir to Staging area|
|git commit -m|code will go from Staging area to Local repo|
|git push|code will go from Local repo to remote repo|
